General

To use this plug-in, Logitech Media Server (henceforth abbreviated as LMS) is required.

Basic Settings

LMS address: Leaving this field empty will force Squeezelite to search for LMS in your network automatically. If you provide a hostname or IP address, below you will get two links to your LMS web interface and LMS configuration. Port is not that from web interface (9000), but LMS stream communication. Usually you don't need to enter a port.

Zones

Active: Every zone can be activated or disabled.
Zone Name: This name is used in LMS and LMS apps.
MAC address: You must provide different MAC addresses for multiple zones. The MAC is the unique identifier in your LMS network.
Roll a MAC address: Generates a random MAC.
Output: Displays all outputs of your LoxBerry.
Additional parameters: Use this field for additional parameters starting Squeezelite.

Add and delete zones

You can add any number of zones by the Add button. You only can delete the last entry a time. The full configuration is saved with adding and deleting.

Save, Save and apply

Save will save your configuration.
Save and apply saves your configuration and restarts all zones with the new configuration.
